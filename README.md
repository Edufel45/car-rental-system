\# 🚗 Car Rental System



A full-featured car rental management system built with Django.



\## Features



\- ✅ Browse available cars with images

\- ✅ Online booking request system

\- ✅ Admin panel for managing cars and bookings

\- ✅ Export bookings to Excel

\- ✅ Responsive design (Bootstrap 5)



\## Tech Stack



\- Django 6.0

\- SQLite3

\- Bootstrap 5

\- openpyxl (Excel export)



\## Installation



```bash

git clone https://github.com/Edufel45/car-rental-system.git

cd car-rental-system

python -m venv venv

venv\\Scripts\\activate

pip install -r requirements.txt

py manage.py migrate

py manage.py createsuperuser

py manage.py runserver
